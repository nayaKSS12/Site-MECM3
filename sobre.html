SOBRE

Este projeto tem como objetivo criar um sistema automatizado de separação de objetos com base em materiais e cores, onde o controle é realizado por um Controlador Lógico Programável (CLP) com supervisório. Além disso, um Arduino será empregado para coletar dados do CLP e enviá-los para um serviço em nuvem, onde será criado um dashboard para monitoramento remoto.

Resumo

Desenvolver um sistema automatizado capaz de separar objetos com base em materiais (metal ou plástico) e cores (preto ou vermelho). Integrar atuadores pneumáticos lineares, sensores, e sistema de controle por CLP com supervisório. Utilizar um Arduino para coletar dados do CLP. Estabelecer a conexão entre o Arduino e a nuvem para enviar dados.
